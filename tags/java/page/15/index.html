Some thoughts on the JBoss AS 5 release

December 18, 2008
JBoss application server 5 GA released after a very long period of silence from “JBoss, a division of Red Hat”. The new version of JBoss application server supports Java EE 5 and is based on modularity concept with a small kernel named JBoss MicroContainer in the core and all functionalities as modules around the kernel. JBoss kernel is POJO based and all modules follows the same principals, some configuration files are required to configure the services which the kernel will load.
